# 个人网站部署指南

> 将 personalWeb 部署到 Vultr 东京 VPS（108.61.162.159）
> 域名：yywtaylor.cyou

---

## 第 0 步：准备工作（在本地电脑操作）

### 0.1 确认项目文件

确保 `C:\baidunetdiskdownload\personalWeb` 目录下有这些文件：
- `.env.production`（已配置好）
- `docker-compose.yml`
- `Dockerfile`
- `deploy/nginx-yywtaylor.cyou.conf`

### 0.2 上传项目到 VPS

打开 PowerShell（在 `C:\baidunetdiskdownload\personalWeb` 目录下），执行：

```powershell
# 将整个项目上传到 VPS
scp -r .\* root@108.61.162.159:/root/personalWeb/
```

> 如果提示输入密码，输入 SSH 密码即可。

---

## 第 1 步：修改 Xray 端口（释放 443）

SSH 登录 VPS：

```bash
ssh root@108.61.162.159
```

登录后执行：

```bash
# 进入 3X-UI 面板设置，修改端口
x-ui settings
```

选择菜单选项修改端口：
- 将 Xray 端口从 **443** 改为 **8443**
- 其他设置不变

重启 3X-UI：

```bash
x-ui restart
```

确认端口已释放：

```bash
ss -tlnp | grep 443
# 应该没有输出（443 已空闲）
```

> **注意**：修改后，你的代理客户端（v2rayN）也需要更新端口为 8443。

---

## 第 2 步：安装 Docker

```bash
# 安装 Docker
curl -fsSL https://get.docker.com | sh

# 验证安装
docker --version
```

---

## 第 3 步：申请 SSL 证书

```bash
# 申请 yywtaylor.cyou 的证书（使用已有的 acme.sh）
acme.sh --issue -d yywtaylor.cyou -d www.yywtaylor.cyou --standalone

# 复制证书到统一位置
mkdir -p /root/cert/yywtaylor
cp -f /root/.acme.sh/yywtaylor.cyou_ecc/fullchain.cer /root/cert/yywtaylor/fullchain.pem
cp -f /root/.acme.sh/yywtaylor.cyou_ecc/yywtaylor.cyou.key /root/cert/yywtaylor/privkey.pem
```

---

## 第 4 步：安装并配置 Nginx

```bash
# 安装 Nginx
apt update && apt install -y nginx

# 复制网站配置
cp /root/personalWeb/deploy/nginx-yywtaylor.cyou.conf /etc/nginx/sites-available/yywtaylor.cyou

# 启用站点
ln -s /etc/nginx/sites-available/yywtaylor.cyou /etc/nginx/sites-enabled/

# 删除默认站点
rm -f /etc/nginx/sites-enabled/default

# 测试配置
nginx -t

# 启动 Nginx
systemctl enable nginx
systemctl start nginx
```

---

## 第 5 步：构建并启动 Docker 容器

```bash
cd /root/personalWeb

# 复制生产环境变量
cp .env.production .env

# 构建并启动
docker compose up -d --build

# 查看运行状态
docker compose ps
docker compose logs -f
```

---

## 第 6 步：配置防火墙

```bash
# 开放网站端口
ufw allow 80/tcp
ufw allow 443/tcp

# 确认规则
ufw status numbered
```

---

## 第 7 步：修改 Cloudflare DNS

在 Cloudflare 控制面板中，为 `yywtaylor.cyou` 添加 DNS 记录：

| 类型 | 名称 | 值 | 代理状态 |
|------|------|-----|---------|
| A | `@` | `108.61.162.159` | **灰云**（仅 DNS） |
| A | `www` | `108.61.162.159` | **灰云**（仅 DNS） |

> **重要**：必须用**灰云**（仅 DNS），不要开启橙色云（代理），否则 acme.sh 申请证书会失败。
